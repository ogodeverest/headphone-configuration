## Run Locally

Clone the project

```bash
  git clone https://github.com/ogodeverest/headphone-configuration.git
```

Go to the project directory

```bash
  cd headphone-configuration
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```
